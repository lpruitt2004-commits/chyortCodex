# Week 4: MySQL & Database Integration (Robin Nixon, Learning PHP, MySQL & JavaScript 5th Edition)

**Theme:** Storing and Retrieving Data with MySQL  
**Duration:** 5 days  
**Goal:** Learn MySQL basics and connect PHP to a database

---

## Learning Objectives

- [ ] Understand relational database concepts
- [ ] Create and manage MySQL tables
- [ ] Perform CRUD operations (Create, Read, Update, Delete)
- [ ] Connect PHP to MySQL (mysqli/PDO)
- [ ] Display database content in HTML

---
