Goal: Initialize a repository, make clean commits, and inspect history.

Tasks:
1) Initialize a new repo in `The-Directory/week9-demo` and create a README.
2) Add a `.gitignore` to ignore `node_modules` and `.env`.
3) Make two atomic commits:

4) Inspect history with `git log --oneline --graph` and `git show`.
5) Make a change to README and use `git diff` to see the change before committing.
6) Practice safe undo: create a file, stage it, then unstage with `git restore --staged`.
