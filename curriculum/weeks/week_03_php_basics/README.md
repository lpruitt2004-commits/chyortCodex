**Theme:** Server-Side Programming with PHP

**Goal:** Learn PHP syntax, control structures, and form handling

## Learning Objectives

- [ ] Understand PHP syntax and variables
- [ ] Use control structures (if, switch, loops)

- [ ] Handle HTML forms with PHP
